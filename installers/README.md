# Installers

## ⚠️ GitHub File Size Limit

Note: The actual installer files (`.dmg`, `.exe`, `.AppImage`) are **not included in this repository's git history** because they exceed GitHub's 100MB file size limit.

### Saved Locally
The installers have been generated and saved locally in this folder:

1.  **macOS**: `DStudiosLab-1.0.0-arm64.dmg`
2.  **Windows**: `DStudiosLab Setup 1.0.0.exe`
3.  **Linux**: `DStudiosLab-1.0.0.AppImage`

### How to Distribute
To share these apps, please upload them to the **GitHub Releases** page:

1.  Go to [https://github.com/dvsaikumar/PromptLab/releases](https://github.com/dvsaikumar/PromptLab/releases)
2.  Click "Draft a new release".
3.  Tag version: `v1.0.0`.
4.  Drag and drop the files from this `installers` folder into the upload box.
5.  Click "Publish release".
